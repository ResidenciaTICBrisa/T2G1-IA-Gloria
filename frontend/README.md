## Como rodar o projeto localmente?

- Clone o repositório:
`git clone <URL_DO_REPOSITORIO>`
- Acesse a pasta backend: 
`cd frontend`
- Copie o arquivo .ev.example para um arquivo .env.
- Edite o arquivo .env com as suas configurações.
- Baixe o Vite:
`npm i vite`
- Baixe as dependências: 
`npm install`
- Execute o projeto:
`npm run dev`
